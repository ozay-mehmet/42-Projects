# 🐍 42 Python Projects

Bu repository, 42 School eğitim sürecinde geliştirilen Python projelerini içermektedir. Amaç, Python programlama dilinin temelinden başlayarak daha ileri seviyelere kadar farklı programlama paradigmalarını uygulamalı olarak öğrenmektir.

Projeler, fonksiyonel programlama yaklaşımından başlayarak nesne yönelimli programlamaya (OOP) kadar ilerleyen bir öğrenme sürecini temsil eder.

---

## 📂 Proje Yapısı

Repository, farklı seviyelere ayrılmış modüllerden oluşur:
```bash
42-Python-Projects/
│
├── Python-Module-0/
├── Python-Module-1/
└── Python-Module-2/
└── ...
```
Her klasör, belirli bir programlama konseptine odaklanır.

---

## 🚀 Python0 – Fonksiyonel Programlama Temelleri

Bu bölüm, Python’un temel yapı taşlarını öğrenmeye odaklanır.

### 📌 İçerik:
- Fonksiyon tanımlama ve çağırma
- Parametre kullanımı
- Return değerleri
- Basit algoritma geliştirme
- Kod tekrarını azaltma (DRY prensibi)

### 🎯 Amaç:
Fonksiyon mantığını kavrayarak daha modüler ve okunabilir kod yazabilmek.

---

## 🧠 Python1 – Nesne Yönelimli Programlama (OOP)

Bu bölümde Python’un nesne yönelimli özellikleri üzerine çalışılır.

### 📌 İçerik:
- Sınıf (class) yapısı
- Nesne (object) oluşturma
- Encapsulation (kapsülleme)
- Inheritance (kalıtım)
- Polymorphism (çok biçimlilik)

### 🎯 Amaç:
Gerçek dünya problemlerini nesne yönelimli yaklaşımla modelleyebilmek.
